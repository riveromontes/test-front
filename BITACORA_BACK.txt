1- Creamos la carpeta

    mkdir test-backend

2- Entramos en la carpeta del proyecto e instalamos

    npm init -y

3- Instalamos Dependencias

    npm install express mysql2 jsonwebtoken bcryptjs dotenv cors nodemon

    3.1- express: Framework para crear rutas y manejar solicitudes HTTP.
    3.2- mysql2: Cliente para conectar a MySQL.
    3.3- jsonwebtoken: Librería para trabajar con JWT.
    3.4- bcryptjs: Para hashear contraseñas.
    3.5- dotenv: Para manejar variables de entorno.
    3.6- cors: Permitir solicitudes del frontend.
    3.7- nodemon: Reiniciar automáticamente el servidor durante el desarrollo.
